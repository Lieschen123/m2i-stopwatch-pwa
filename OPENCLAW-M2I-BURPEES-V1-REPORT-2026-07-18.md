## Plan of attack

1. Extend `createChallengePlan` in `src/challenge.js` with optional `activityType`, `scoringModel`, `durationSeconds`, `minReps` — defaults preserve movement behavior.
2. Extend `workoutMeetsChallenge` with a burpee branch keyed off `challenge.activityType === 'burpees'`.
3. Add optional burpee fields (`activity_type`, `scoring_model`, `proof_type`, `rep_count`) to `createClaim` in `src/claim.js`. Existing calls unchanged.
4. Extend `createWorkout` in `src/stopwatch.js` with optional activity type + challenge target duration seconds.
5. UI in `src/main.js`:
   - Home form: `Movement / Run` vs `Burpees` toggle; when Burpees, show a duration input (default 7 min).
   - Challenge card shows burpee summary.
   - Start workout for burpee challenge → workout screen counts up; Finish prompts for reps before signing.
   - Claim screen + history + board display reps for burpee claims.
6. Tests: add burpee plan/claim/validity coverage.
7. Run `npm test` and `npm run build`.
